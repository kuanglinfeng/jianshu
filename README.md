# jianshu

使用react全家桶开发简书网部分功能

1. 使用`styled-components`书写组件样式

2. 使用`immutable.js`管理store中的数据

3. 使用`redux-immutable`统一数据格式

4. 使用`redux-thunk`管理异步action

5. 使用`axios`请求异步数据

6. 使用`react-router-dom`书写路由

## 预览

1. git clone git@github.com:kuanglinfeng/jianshu.git
2. cd jianshu 
3. npm install
4. npm run start
